### **--> Python Variables \& Naming Conventions**

* variables are container for storing data
* Assign values using the = sign
* Python automatically detects the type unlike other languages
* Variables can hold numbers, text, \& other data types
* Python is dynamically typed, so there is no need to declare variable types.
* Use descriptive variable names e.g., studentName, total\_marks(if you are write long code or building projects)
* Not start with a number , start with a letter or \_
* no spaces , no keywords
* Types(Integers, floats, Strings, Booleans, )

### **--> Defining \& Calling Functions**

* A function is a reusable block of code that performs a specific task
* increase reusability of code
* helps make programs organized

### **--> Variable Scope in Python - Local vs Global**

* **Local variable:** A variable declared inside a function is called a local variable
* **Global Variable:** A variable declared outside the function(in the main program) is called global variable





### **--> Lambda Functions in Python**
